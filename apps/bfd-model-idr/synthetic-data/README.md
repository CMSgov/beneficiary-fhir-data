# v3 BFD (Partial) Synthetic Dataset

> [!IMPORTANT]
> The claims data is **too large** to store in the Git repository as several of the files exceed GitHub's file limit. As a result, those files (the _entire_ dataset) are stored in Box. Consult with the BFD team in Slack for more detail.

This directory contains the full beneficiary, contract, and provider history v3 BFD synthetic dataset in form of the IDR schema. This synthetic dataset is loaded into all 3 of BFD's enduring environments, and is randomly-generated using the Synthetic Data Generators such that there is no PHI/PII within any of them.
