# `repack` Terraservice

A small Terraservice for spinning up a `t4g.micro` EC2 Instance intended to be used, on-demand, for running `pg-repack` on the corresponding environment's Aurora Postgres database.

Currently, the EC2 Instance is a simple AL2023 AMI with no customization. The operator is expected to install `pg-repack` and other facilities for running `pg-repack` as this operation is not a common one for BFD.
